What is Lelylan Types

Lelylan Types is a place where you can define and discuss the structure of any device type like lights, locks, cameras, sensors and more. Why? Because to create an open connected home where everything communicates we need an open and clear way to define how these devices work.

Right now we have dozens of standards: Wifi, Zigbee, Ethernet, Konnex, X10 to name some. If you want to let every device communicate between each other you will have hard time. This is one of the reasons why the connected home is not yet a reality. Our solution to this problem is to create a type, which defines a device structure in a unique way through the combination of three key elements:

  • Properties - what vary during time in a device.
  • Functions - what a device can do.
  • Statuses - what a device is in a specific moment.

Follows a simple example describing a basic light.

  • Property status accepting values on and off.
  • Functions turn on and turn off.
  • Statuses light on and light off.

Call for action
