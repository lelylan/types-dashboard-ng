How to create a type

To create a new type you need to click "create type". If signed in, a form asking you for a name and a description will show up.

Properties

Once you have filled the required information you are asked to create the type properties. For a light they could be status, intensity, color and blinking times.

To add a new property click "create new property". A modal form will appear asking for the following params.

Name A descriptive name for the property
Type Property accepted values as text, number, range and color.
Default Value assigned to the device property when initialized.
Suggested List of value/description pairs (visible when type is not set to range). Learn more on dev center.
Range Accepted values defined between a min, max and step value (visible when type is set to range).
